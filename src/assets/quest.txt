Задача:

Напишите задачу, в которой пользователь должен использовать данные из JSON-файла в своем проекте на Vue.js 3. Задача состоит в отображении списка фильмов на странице.

JSON-файл: Создайте JSON-файл movies.json, содержащий информацию о фильмах. Каждый фильм должен иметь следующие поля:

title (название фильма)
year (год выпуска фильма)
description (описание фильма)

Отображение данных: Отобразите список фильмов на странице. 
Создайте компонент MovieList, который будет отображать каждый фильм в виде карточки на странице. 
В каждой карточке должны быть отображены название, год выпуска и описание фильма.

Внедрение в проект: Импортируйте JSON-файл movies.json в компонент MovieList и прочитайте данные из файла. 
Создайте цикл с помощью директивы v-for, чтобы пройти по каждому фильму в списке и отобразить его на странице. 
Передайте данные фильма в компонент MovieCard, который будет отображать информацию о каждом фильме.

Дополнительные требования: Реализуйте возможность сортировки списка фильмов по году выпуска или по названию.
 Предоставьте пользователю интерфейс (например, кнопки или выпадающий список) для выбора определенного порядка сортировки.
  При изменении порядка сортировки список фильмов должен обновляться автоматически.

Обратите внимание: Данные из JSON-файла могут быть доступны через импорт или через API-запрос,
 в зависимости от вашей реализации проекта.
